Movies API

Objective:
Develop a Node.js Web API using an SQLite database for CRUD (Create, Read, Update, Delete) operations on movies.

The Movie model should include the following attributes:

Title (string): The title of the movie, required, max 150 characters.
Synopsis (text): A summary of the movie, required, max 1500 characters.
Genre (string): The type of the movie (e.g., romance, science fiction), required, max 150 characters.
Director (string): The person who directed the movie, required, max 150 characters.
Release Date (date only): The date the movie was released, required.
Rating (decimal): The average rating given to the movie, optional, between 1 to 10, with 1 decimal place.
